# ProfitStack Release Checklist

- [ ] env vars set in host
- [x] app starts with `npm start`
- [x] `/api/health` returns 200
- [x] login page loads
- [x] session endpoint resolves the intended user/org
- [x] dashboard loads live Supabase data
- [x] CRM page loads
- [x] overrides load and save
- [x] sync page loads
- [x] logout works
- [ ] no secrets committed

## Scope note
- This checklist supports a **controlled first-tenant launch**, not a public self-serve release.
