# ProfitStack Frontend Page Map

## Core pages
- `/login`
- `/dashboard`
- `/settings/crm`
- `/settings/overrides`
- `/settings/account`

## Page purposes
### /login
User access into the app.

### /dashboard
Primary operating view for current week, last week, and next 3 weeks.

### /settings/crm
Connect and monitor Housecall Pro or future CRMs.

### /settings/overrides
View and manage trusted manual overrides.

### /settings/account
Organization profile, timezone, and admin settings.

## Rule
Dashboard first. Settings pages support trust, connection, and control.
