Chrome helper setup

Before you connect Housecall Pro

This step makes the Chrome-only connection path clear before the real CRM connection happens. It is the right place for the extension install step and keeps clients out of the technical page too early.

1. Use Google Chrome

This connection only works in Google Chrome. Other browsers are not supported for the Housecall Pro helper flow.

2. Keep both tabs in the same profile

Open The Nut Report and Housecall Pro in the same Chrome profile. That is how the helper knows which account to connect.

3. Install the Chrome helper, then connect

Install the helper from the Chrome Web Store, then continue into the Housecall Pro connection step.
