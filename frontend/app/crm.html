Desktop connect flow

1. Stay logged into The Nut Report

Keep this Nut Report tab open in Chrome. The helper uses this tab to know which account to connect.

2. Log into Housecall Pro

Use the top button to go straight into Housecall Pro and make sure the client login is active in the same Chrome profile.

3. Click the Chrome helper

Use the helper to capture the active HCP session and connect it automatically to this account.

Then do this
Come back to the dashboard and click Refresh Data. If Housecall Pro logs the client out later, reconnect the same way. On Sunday night, the dashboard may already roll into the next week if your reporting week starts Monday.
If the helper does not connect
Make sure you are in Google Chrome, logged into Housecall Pro, and using the same Chrome profile for both tabs. If needed, use the manual fallback below or email outsidethebusinessbox@gmail.com.
